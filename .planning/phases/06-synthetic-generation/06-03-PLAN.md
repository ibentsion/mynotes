<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Network → local filesystem | Font files downloaded from GitHub raw / Google Fonts CDN written to assets/fonts/ |
| --manifest / --wordlist files → process | User-supplied CSV/text read at runtime |
| Generated PNGs → training pipeline | Synthetic crops consumed unchanged by CropDataset/load_crop |
| Process → ClearML server | Args + manifest artifact + per-char scalars uploaded |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-07 | Tampering | font bytes from remote URL | mitigate | `requests.get(...).raise_for_status()`; URLs are HTTPS pinned constants verified in RESEARCH.md (Sources); fonts written under assets/fonts/ only; .ttf gitignored so a bad download cannot silently enter the repo |
| T-06-08 | Spoofing | font CDN/raw URL serving unexpected content | accept | OFL fonts from Google Fonts + AlefAlefAlef GitHub; integrity not hash-pinned (acceptable for a local personal tool; manual `--fonts_dir` override available if a URL is compromised, RESEARCH.md A3) |
| T-06-09 | Denial of Service | TRDG returning None forever (Pitfall 3) starving --count | mitigate | render loop continues on None and keeps sampling new texts until `--count` crops saved; dark default text (#282828) on white maximizes contrast to minimize None rate |
| T-06-10 | Tampering | synthetic crops poisoning the training set unnoticed | mitigate | crops written ONLY to `--output_dir` (default outputs/synthetic, gitignored); manifest status hard-coded `labeled` but path is a distinct directory the operator opts into via `--pretrain_manifest` in Phase 7 — never merged into the real manifest |
| T-06-11 | Repudiation | which args/corpus produced a synthetic set is unknown | mitigate | init_task + task.connect(vars(args)) + manifest artifact upload + git commit hash (clearml_utils convention) make every run reproducible and attributable (D-07) |
| T-06-12 | Information Disclosure | personal Hebrew text uploaded to ClearML | accept | Only aggregate per-char counts + the synthetic manifest (derived/recombined words) are uploaded; consistent with existing train_ctc.py ClearML usage and PROJECT.md local-data posture; no raw source pages leave the machine |
</threat_model>
