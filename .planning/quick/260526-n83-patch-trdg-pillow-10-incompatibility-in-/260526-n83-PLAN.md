Insert exactly the following block (one-line comment explains why, then the conditional patch):

```python
# Pillow 10 removed ImageFont.FreeTypeFont.getsize; trdg 1.8.0 still calls it. Restore via getbbox.
from PIL import ImageFont

if not hasattr(ImageFont.FreeTypeFont, "getsize"):
    def _compat_getsize(self, text, *args, **kwargs):
        bb = self.getbbox(text)
        return bb[2] - bb[0], bb[3] - bb[1]

    ImageFont.FreeTypeFont.getsize = _compat_getsize  # type: ignore[attr-defined]
```
