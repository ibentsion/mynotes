# Quick Task 260526-n83: Patch trdg / Pillow 10 getsize incompatibility

**One-liner:** Conditional `getsize` shim on `ImageFont.FreeTypeFont` via `getbbox` unblocks trdg 1.8.0 on Pillow 12.

## What Was Done

Added a Pillow 10+ compatibility shim to `src/generate_synthetic.py`. trdg 1.8.0 calls `image_font.getsize(text)` which Pillow removed in v10. The shim restores the method only when absent, delegating to `getbbox` to compute `(width, height)`.

## Shim Location

Inserted after the third-party import block (line ~14), before the `# Constants` header. `from PIL import ImageFont` was placed in the isort-correct position (third-party block), with the conditional assignment block immediately below all imports.
