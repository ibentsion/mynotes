## Training-Run Analysis (UK2-01)

Symptoms from the most recent CPU run:

| Signal               | Trajectory                  | Reading                                            |
|----------------------|-----------------------------|----------------------------------------------------|
| train_loss           | 1.05 -> 0.20 over 30 epochs | Model memorises the train split.                   |
| val_loss             | 3.06 -> 4.28                | Severe overfitting; gap widens monotonically.      |
| val_cer              | best 0.5966 @ epoch 20      | Plateau, then drift up. Classic early-stop signal. |
| blank_frac           | ~0.94 throughout            | Bias fix helped vs pre-fix but blank still wins.   |
| inf_loss count       | 2 -> 5 across training      | A few train sequences are still longer-than-T.     |
| sample preds         | `ניהול`->`ללל`, `השיחות`->`השתות`, `עם`->`עם` | Recognises short or first-char patterns; loses interior structure. |

What this means:

1. **Capacity >> data.** 30 epochs of train at lr=1e-3 with rnn_hidden=256 x 2 layers on
   ~100 labeled crops drives train loss into the floor while val never catches up. The
   model is fitting handwriting *identity* of training samples, not Hebrew script.
2. **Blank collapse is suppressed, not solved.** blank_frac=0.94 means even with the
   `-2.0` bias init the easiest path through CTC is still mostly-blank. Short labels
   (e.g. `עם`) escape because they tolerate high blank fraction; longer words don't.
3. **`inf_loss` drift = CTC infeasibility.** Some `T = W//4` is shorter than the label
   length for the augmented copies that get rotated/cropped. Not catastrophic
   (`zero_infinity=True` zeros them) but indicates we're losing gradient on the hardest
   training examples — exactly the ones we need.
4. **First-character bias in preds.** `השיחות`->`השתות` keeps prefix, mangles middle.
   Consistent with overfit CNN features + RNN that hasn't seen enough context to model
   character transitions inside words.

Suggested next experiments (ordered by cost/value, **not part of this task's code change**):

| # | Change                                                | Hypothesis                                              | Cost |
|---|-------------------------------------------------------|---------------------------------------------------------|------|
| A | Early-stopping patience=5 on val_cer + restore-best   | Stops drift after epoch 20; "free" win                  | trivial |
| B | Increase `aug_copies` 4->8, add slight shear/blur     | Train set x2, narrows train/val gap                     | trivial |
| C | Add weight decay (1e-4) on Adam                       | Mild regularisation; orthogonal to aug                  | trivial |
| D | Shrink model: rnn_hidden=128, num_layers=1            | Cuts capacity to match data size                        | trivial |
| E | LR schedule (cosine or ReduceLROnPlateau on val_cer)  | The flat 1e-3 plateaus at epoch 20; LR drop often helps | trivial |
| F | Label smoothing / focal-CTC variant                   | Penalise overconfident blank predictions                | medium |
| G | Pad input width so `T >= max_label_len + 2`           | Eliminates `inf_loss` cases entirely                    | medium |
| H | More labeled data (the real fix)                      | This dataset is fundamentally too small                 | large |

Pick A+B+D for the next sweep before reaching for anything else. The Optuna tuning
infrastructure from Phase 05 is the right vehicle.

    ```python
    def compute_char_saliency(
        model: CRNN,
        charset: list[str],
        device: torch.device,
        crop_path: str,
    ) -> tuple["np.ndarray", str, "np.ndarray"]:
        """Grad-CAM on the last CNN conv block for non-blank predicted timesteps.

        Returns (crop_hw, pred_text, saliency_hw), all H x W with saliency in [0, 1].
        Hooks the last Conv2d in model.cnn. Caller does NOT need no_grad - this
        function manages eval mode + autograd internally.
        """
        import numpy as np  # noqa: PLC0415
        import torch.nn.functional as F  # noqa: PLC0415

        image = load_crop(crop_path).unsqueeze(0).to(device)  # (1, 1, 64, W)
        w = image.size(3)
        padded_w = math.ceil(w / 4) * 4
        if padded_w != w:
            pad = torch.zeros(1, 1, image.size(2), padded_w, device=device)
            pad[:, :, :, :w] = image
            image = pad

        was_training = model.training
        model.eval()  # batchnorm/dropout off, autograd still on

        target_layer = None
        for m in reversed(list(model.cnn)):
            if isinstance(m, nn.Conv2d):
                target_layer = m
                break
        if target_layer is None:
            raise RuntimeError("CRNN.cnn has no Conv2d layer")

        activations: list[torch.Tensor] = []
        gradients: list[torch.Tensor] = []
        h_fwd = target_layer.register_forward_hook(
            lambda _m, _i, o: activations.append(o)
        )
        h_bwd = target_layer.register_full_backward_hook(
            lambda _m, _gi, go: gradients.append(go[0])
        )
        try:
            for p in model.parameters():
                p.requires_grad_(True)
            logits = model(image)  # (T, 1, C)
            log_probs = logits.log_softmax(2)
            pred_indices = greedy_decode(log_probs[:, 0, :])
            pred_text = "".join(charset[i - 1] for i in pred_indices)

            argmax_per_t = log_probs[:, 0, :].argmax(dim=1)  # (T,)
            non_blank_mask = argmax_per_t != 0
            if non_blank_mask.any():
                gathered = log_probs[:, 0, :].gather(
                    1, argmax_per_t.unsqueeze(1)
                ).squeeze(1)
                score = gathered[non_blank_mask].sum()
                model.zero_grad(set_to_none=True)
                score.backward()

            if activations and gradients:
                act = activations[0]   # (1, C', H', W')
                grad = gradients[0]    # same shape
                alpha = grad.mean(dim=(2, 3), keepdim=True)  # (1, C', 1, 1)
                cam = F.relu((alpha * act).sum(dim=1, keepdim=True))
                cam = F.interpolate(
                    cam, size=image.shape[-2:], mode="bilinear", align_corners=False
                )
                sal = cam[0, 0].detach().cpu().numpy()
                rng = float(sal.max()) - float(sal.min())
                sal = (
                    (sal - float(sal.min())) / rng
                    if rng > 1e-8
                    else np.zeros_like(sal)
                )
            else:
                sal = np.zeros(image.shape[-2:], dtype=np.float32)
        finally:
            h_fwd.remove()
            h_bwd.remove()
            if was_training:
                model.train()
            model.zero_grad(set_to_none=True)

        crop_hw = image[0, 0].detach().cpu().numpy()
        return crop_hw, pred_text, sal.astype("float32")
    ```
